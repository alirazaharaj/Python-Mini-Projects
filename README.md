# Python Mini Projects Collection

A curated collection of functional Python scripts and mini-applications ranging from core programming logic problem-sets to full Graphical User Interfaces (GUIs) built with Tkinter.

## 🚀 Projects Overview

| # | Project Name | Description | Tech Used |
|---|---|---|---|
| 1 | **Email Validator** | Validates email syntax using Regular Expressions (Regex). | `re` |
| 2 | **Basic Practice Problems** | 12 fundamental programming logic scripts (Factorial, Prime check, etc.). | Core Python |
| 3 | **Shutdown Button GUI** | A Tkinter dashboard to restart, logout, or shutdown your PC. | `tkinter`, `os` |
| 4 | **Typing Speed Tester** | CLI tool that tests your typing speed and calculates character errors. | `time`, `random` |
| 5 | **Network Speed Tester** | Multi-threaded GUI application that tracks upload/download speeds. | `tkinter`, `speedtest-cli`, `threading` |
| 6 | **Text to Handwriting Converter** | Converts standard digital string text into a handwritten image file. | `pywhatkit` |
| 7 | **YouTube Video Downloader** | Script outlines for downloading single videos or full playlists. | `pytube` (Draft) |
| 8 | **Desktop Notifier** | background loop utility that triggers native OS desktop break reminders. | `plyer`, `time` |
| 9 | **Digital Clock** | A visual desktop digital clock showing live time, date, and day details. | `tkinter`, `datetime` |
| 10| **Hangman Game** | Classic interactive word-guessing game played directly via command-line. | `random` |
| 11| **Voice Assistant Tools** | A script featuring text-to-speech synthesis and speech-to-text recognition modules. | `pyttsx3`, `speech_recognition` |
| 12| **Website Blocker** | An automation script that blocks specific distracting websites by modifying the OS hosts file during working hours. | `datetime`, `time` |
